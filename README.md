Spelling Checker

Reads a text file, checks each word against a dictionary, and writes two output files.

	•	Matches the required assignment output format
	•	Includes project report PDF

	•	book.txt
	•	words.txt
	•	correct_words.txt
	•	misspelled_words.txt
	•	Homework 7 - Spelling Project Report.pdf
